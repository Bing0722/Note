## 一、Linux常用操作

### 0、用户和用户组

```bash
# 添加新用户
```

### 4、文件权限和所有权

```bash
# 改变文件或目录的权限
chmod
# 1.chmod 基本概念
文件或目录的权限由三组表示:    
    u(所有者), g(组), o(其他人) 
每组权限有:
    r(读), w(写), x(执行)
# 2.使用数字模式更改权限
    r = 4, w = 2, x = 1
数字组合表示权限:
    7(rwx): 读、写、执行 (4 + 2 + 1)
    6(rw-): 读、写 (4 + 2)
    5(r-x): 读、执行 (4 + 1)
    4(r--): 读 (4)
    3(-rx): 写、执行 (2 + 1)
    2(-r-): 写 (2)
    1(--x): 执行 (1)
实例: chomd 755 filename 设置所有者权限为读、写、执行，组和其他人权限为读、执行
# 3.使用符号模式更改权限
符号模式使用符号来修改权限
    u(所有者) g(组) o(其他人) a(所有用户等于ugo)
权限符号:
    + :添加权限
    - :移除权限
    = :赋予确切的权限
实例: chmod u+x filename 给文件的所有者添加执行权限

# 改变文件或目录的所有者
chown

# 改变文件或目录的所属组
chgrp

```

### 5、压缩和解压缩

```bash
# 打包和压缩文件
tar
# 1.tar 常用选项
    -c :创建归档文件
    -x :解压归档文件
    -v :显示详细信息 即显示处理的文件列表
    -f :指定归档文件的名称
    -t :列出归档文件内容 而不解压
# 2.压缩和解压
## 2.1使用gzip
    创建并压缩归档: tar -czvf 归档名.tar.gz 文件/目录
    示例: tar -cvzf archive.tar.gz file.txt dir1 会创建一个gzip压缩的归档文件 archive.tar.gz
    解压gzip压缩的归档: tar -xzvf 归档名.tar.gz
    tar -xzvf archive.tar.gz 会解压缩 archive.tar.gz

# 压缩文件
gzip

# 解压缩文件
gunzip

```

### 6、系统信息和监控

```bash
# 显示系统信息
uname

# 显示系统中的进程
top

# 更友好的top命令 以彩色显示进程信息
htop

# 显示当前进程的状态
ps
    -elf :以长格式显示系统中所有进程的详细信息
        -e :显示系统中所有进程
        -l :以长格式显示 提供比默认输出更详细的信息
        -f :显示完整格式的输出 包括进程的父进程ID、用户ID、启动时间等
    -elLf :以长格式显示系统中所有线程的详细信息


# 显示内存使用情况
free

# 报告文件系统的磁盘空间使用情况
df
    -h :以 1000 作为基数 以 SI 单位显示输出结果(例如，MB、GB)

# 检查文件或目录的磁盘使用量
du
    -h :以 1000 作为基数 以 SI 单位显示输出结果(例如，MB、GB)

```
